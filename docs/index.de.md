# SignalRGB Desktop-Wallpaper

**Live-RGB-Glow auf deinem Desktop, gesteuert von deinem SignalRGB-Effekt.**

Mehrere Monitore · Einstellungen pro Bildschirm · Ein-Klick-Installer · Lively
*und* Wallpaper Engine.
