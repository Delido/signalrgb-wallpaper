OpenRGB output (beta)

Mirror the wallpaper's live glow colour onto your OpenRGB- controlled hardware (RAM, fans, keyboards, …). Requires OpenRGB running with its SDK Server enabled (Settings → SDK Server in OpenRGB).
—

Device positions on live preview

Drag each marker onto the spot on the wallpaper whose colour that device should follow. Default: centre.

Colour source per screen (beta)

Picks where each screen's live glow colour comes from. SignalRGB is the default — the wallpaper follows whatever effect SignalRGB is running. Switch a screen to OpenRGB to mirror the current LEDs of an OpenRGB device, or to sACN to listen for an incoming E1.31 universe.

sACN / E1.31 output (beta)

Streams the per-screen averaged glow colour as DMX over IP (E1.31 / sACN) on the standard multicast group. Pick this up with xLights, QLC+, Hyperion, or any hardware DMX / sACN controller. First three DMX channels of each universe carry R / G / B.
—

Home Assistant / MQTT bridge (beta)

Publishes per-screen state to your MQTT broker and accepts control commands on the matching `*/set` topics. Drop-in for Home Assistant, Node-RED, IFTTT-style automations.

REST API token (beta)

Auto-generated per install. Loopback requests bypass auth so the Configurator + same-host integrations work without configuration. External clients send it as `Authorization: Bearer <token>`. OpenAPI spec at /api/openapi.json.
View OpenAPI spec

3rd-party widget plugins (beta)

Drop a folder with a manifest.json under %LOCALAPPDATA%\SignalRGBWallpaper\plugins\ and hit Rescan. Each plugin runs in a sandboxed iframe on the wallpaper page. Contract documented in docs/plugin-api.md.
▾

Per-app profiles

When a matching .exe is focused, swap the chosen screen's preset slot automatically. Reverts to the previous state when the foreground changes away. One rule active at a time — order in the list = priority on conflict.
▾

Backup & Restore

Export every setting plus the wallpaper library and per-screen backgrounds as one ZIP. Restore drops the ZIP back into %LOCALAPPDATA%\SignalRGBWallpaper and reloads — useful for fresh installs, multi-PC setups, or "try something risky" recovery.
